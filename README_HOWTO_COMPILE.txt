2) Установить расширение MCUXpresso for VS Code
3) Установить Cmake расширение для VS Code
4) Настроить конфиги json в .vscode. Само расширение так не умеет.
6) В расширении CMake(Configure) выбрать способ компиляции ARM GCC
7) В расширении MCUXpresso for VSCode компилируем
